14. Напишите программу которая, попросит ввести строку и выведет True если введенное строка палиндром иначе False

> Палиндром – любой текст, который одинаково читается в обе стороны

```
Пример:
Задача 14 (палиндром)

Введите слово: level
True

Введите слово: dom
False

Введите слово: house
False

Введите слово: kabak
True
```
